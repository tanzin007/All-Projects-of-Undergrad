# Smart Budget & Expense Tracker with Insights

An eye-catching financial analytics dashboard utilizing high-speed data manipulation engines to monitor transactions and project budget trends.

## 🚀 Architectural Features
- **Pandas Manipulation Matrix:** Utilizes the Python `Pandas` engine to execute rapid data aggregation and runtime grouping routines.
- **Predictive Spending Logic:** Implements mathematical trend analysis across monthly log frames to generate real-time alerts.
- **Zero-Database JSON Storage:** Saves transactions in flat structures, completely bypassing local database configurations.

## 🛠️ Tech Stack
- **Backend Frame:** Python / Flask
- **Data Engine:** Pandas
- **Visualization Mapping:** Chart.js (Frontend CDN)

## ⚙️ Local Setup
1. Open folder in VS Code terminal and install standard pip dependencies:
   ```bash
   pip install flask pandas
   ```
2. Run the platform engine:
   ```bash
   python app.py
   ```

# All Projects of Undergrad (BSc in CSE)

This repository contains all the academic and personal projects I developed while completing my **BSc in Computer Science and Engineering (CSE)** at **Khwaja Yunus Ali University** (Duration: July 2022 to July 2026).

---

## 📂 Repository Structure & Project Directory

Each folder represents a specific academic year and semester, containing tailored projects and their individual documentation (`readMe.md`).

### 🎓 1st Year
*   **1st Semester (1.1)**
    *   `bank_account_simulator` — Basic banking system in C++.
    *   `departmental_store_utility` — Inventory and utility management tool.
    *   `hotel_room_booking_system` — Room booking management software.
    *   `library_management_system` — Books and library log keeper.
*   **2nd Semester (1.2)**
    *   `Card Game System` — Interactive card gameplay engine.
    *   `Text-Based Role Playing Game` — CLI-based story RPG game.
    *   `Tic-Tac-Toe Game` — Classic two-player game.
    *   `Vehicle Rental System` — Car/Vehicle booking solution.

### 🎓 2nd Year
*   **3rd Semester (2.1)**
    *   `Hospital Ticket System` — Patient queue and ticket tracker in Python.
    *   `Maze Solver` — DSA-based pathfinding visualization.
    *   `Social Network Connection Finder` — Graph-based friend network analysis.
    *   `Sorting Algorithm Visualizer` — Bubble Sort vs Selection Sort animation.
    *   `Text Editor with Undo-Redo` — Stack/Queue-based text editor utility.
*   **4th Semester (2.2)**
    *   `Retail Management Backend System` — Database and backend API setup (Flask/SQL).
    *   `Student_Tracker` — Academic performance tracker application.

### 🎓 3rd Year
*   **5th Semester (3.1)**
    *   `Book Recommendation` — ML-based collaborative/content filtering system.
    *   `NoteKeeper V1` — Full-stack web note-taking app (HTML/Node.js).
    *   `Smart Budget & Expense Tracker` — Finance dashboard with insights.
*   **6th Semester (3.2)**
    *   `Air Quality Index (AQI) Forecaster` — Deep time-series forecasting models containing 4 sub-projects:
        *   *Europe Air Quality Forecaster* — AQI prediction for European regions.
        *   *US Metropolitan AQI Predictor* — AQI prediction for USA cities.
        *   *Transatlantic Air Quality Forecaster* — Combined model for both Europe and USA.
        *   *Global Metropolitan AQI Predictor* — Global scale prediction covering 50 cities and 14 countries.
    *   `Customer Segmentation` — Unsupervised clustering for market analysis.
    *   `Email Spam & Phishing Classifier` — NLP text classification model.
    *   `Fine-tuning BERT model` — Sentiment Analysis using advanced transformer architecture.
    *   `Vehicle License Plate Detection` — Real-time computer vision detection using YOLOv8.
    *   `Waiter_Robot` — Robotics core control logic and ML scripts.

### 🎓 4th Year
*   **7th Semester (4.1)**
    *   `Custom-CNN-From-Scratch-Using-Framework` — PyTorch heavy CNN implementation.
    *   `Custom-CNN-From-Scratch-Using-NumPy` — Pure mathematical matrix-level CNN without frameworks.
    *   `Custom_SLM_25M-Parameter` — Small Language Model trained on TinyStories from scratch.
*   **8th Semester (4.2)**
    *   `Text to Image - Fine_Tuned` — Stable Diffusion v2.1 LoRA training implementation.
    *   `Text to Image - Scratch` — Generative AI models built from scratch containing 2 sub-projects:
        *   *GAN Model* — Custom 6M Conditional GAN implementation.
        *   *Diffusion Model* — Custom 53M UNet Diffusion model architecture.

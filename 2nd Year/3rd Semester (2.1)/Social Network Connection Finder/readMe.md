# Social Network Connection Finder (Python)

An optimization simulation mimicking platform logic (like LinkedIn degrees of separation) using an Adjacency List Graph representation and BFS traversal.

## 🚀 Algorithm Execution
* **Graph Modeling**: Users are represented as graph vertices, and friendship status forms unweighted, bidirectional graph edges.
* **Shortest Path Strategy**: Utilizing a First-In-First-Out (FIFO) queue via Breadth-First Search ensures the engine computes the absolute minimum degrees of relationship separation.

# Live Sorting Algorithm Visualizer (Python)

An educational console visualizer that animates the classic Bubble Sort algorithm step-by-step using custom ASCII bar blocks (`█` / `░`) inside the terminal.

## 🚀 Concept Breakdown
* **In-Place Sorting**: Operates directly on the data array, using memory-efficient constant space $O(1)$.
* **Visual Anchors**: Highlights the active pointers/indices being compared in real-time, helping students grasp $O(N^2)$ algorithm executions visually.
